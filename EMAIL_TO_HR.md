## 📋 Project Overview

GolfGive is a subscription-driven platform that combines:
- Golf performance tracking
- Monthly prize draw system (lottery-style)
- Charitable giving integration
- Payment processing with Stripe
- Real-time data management

## 🛠️ Technical Stack

**Frontend**:
- Next.js 15 (React Framework)
- TypeScript
- Custom CSS with modern animations

**Backend**:
- Next.js API Routes
- Supabase (PostgreSQL database)
- Supabase Authentication
- Row Level Security (RLS)

**Integrations**:
- Stripe Payment Gateway (INR currency)
- Real-time subscriptions
- Webhook handling

**Features Implemented**:
- User authentication (signup/login)
- Role-based access control (Visitor, Subscriber, Admin)
- Subscription management (Monthly ₹799, Yearly ₹7,999)
- Golf score tracking system
- Automated prize draw system
- Charity selection and contribution tracking
- Admin dashboard for platform management
- Responsive design for all devices
- Collapsible sidebar navigation
